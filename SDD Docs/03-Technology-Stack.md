# Technology Stack

| Layer | Technology | Purpose |
| --- | --- | --- |
| Frontend | Angular | SPA user interface |
| Runtime | Node.js | Execution environment |
| Framework | NestJS | API, DI, modules |
| Language | TypeScript | Type safety |
| ORM | TypeORM | Data access and schema |
| Database | PostgreSQL | Primary data store |
| Auth | JWT | Auth tokens |
| Config | @nestjs/config | Env configuration |
| Validation | class-validator, class-transformer | Request validation |
| Email | Nodemailer | SMTP email |
| PDF | Puppeteer | PDF generation |
| QR | qrcode | QR image creation |
| Excel | ExcelJS | Excel report creation |
| Cloud | AWS ECS, RDS, S3 | Hosting and storage |
| Tooling | eslint, prettier, jest | Quality and testing |
| Patching | patch-package | Dependency patching |
